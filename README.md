# Chiki - Python Voice Assistant

## Overview

Chiki is a simple voice-controlled virtual assistant developed in Python. It listens to user voice commands, processes them using speech recognition, and responds with voice feedback. The assistant can perform various everyday tasks such as opening websites, launching applications, providing date and time information, and more.

## Features

* Voice command recognition
* Text-to-speech responses
* Open YouTube
* Open Google
* Open Spotify
* Open WhatsApp
* Open ChatGPT
* Open Google Maps
* Open Google Translator
* Open Calculator
* Open Notepad
* Open Camera
* Display current date and time
* Open weather updates
* Open news websites
* Exit assistant with a voice command

## Technologies Used

* Python
* pyttsx3 (Text-to-Speech)
* SpeechRecognition
* Google Speech Recognition API
* Webbrowser Module
* OS Module
* Datetime Module

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/chiki-voice-assistant.git
```

2. Install required packages:

```bash
pip install pyttsx3 SpeechRecognition pyaudio
```

3. Run the project:

```bash
python jarvis.py
```
